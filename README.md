# 🔍 OSINT Investigation: Google Dorks for Username Enumeration

An interactive, responsive, web-based presentation deck mapping the **digital footprint of a target username** using public search engines.  
This slide deck features a fully functioning **100 Facebook OSINT Dorks Console** designed to assist cyber intelligence investigators in finding, correlating, and mapping online profiles.

## 🔬 Facebook OSINT Dork Categorization (100 Queries)
The interactive dashboard (Slide 5) organizes queries into **10 modules**:

1. **General OSINT (1–10)**: Profile indexes, friend lists, general posts.  
2. **User Profiles (11–20)**: Bios, follower counts, check-ins.  
3. **Page Details (21–30)**: Organization profiles, contact info, shop listings.  
4. **Group Information (31–40)**: Discussions, join vectors, pinned posts.  
5. **Posts & Media (41–50)**: Geotagged photos, videos, likes, comments.  
6. **Events & RSVPs (51–60)**: Event schedules, guest lists, tickets.  
7. **Reviews & Comments (61–70)**: Ratings, nested comments, feedback.  
8. **Pages & Orgs (71–80)**: Corporate statements, addresses, team registers.  
9. **Ads & Campaigns (81–90)**: Campaigns, spending metrics, link targets.  
10. **Security & Settings (91–100)**: Public settings, MFA indicators, connected apps.
